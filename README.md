# SMME Funding AI Platform

A comprehensive platform for Small, Medium & Micro Enterprises (SMME) funding opportunities and donor connections.

## Quick Start with GitHub Codespaces

1. Click the "Code" button on GitHub
2. Select "Codespaces" tab
3. Click "Create codespace on main"
4. Wait for the environment to load

## Local Development

```bash
npm run install-deps
npm start
```

The application will be available at `http://localhost:3000`

## Project Structure

- `SMME-Funding-AI/` - Main application files
- `index.html` - Landing page
- `server.js` - Express server
- `api.js` - API endpoints

## Features

- Funding opportunity discovery
- Donor portal
- Application management
- User profiles and authentication
